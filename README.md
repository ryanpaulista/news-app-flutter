# news-app-flutter

News app with flutter

## Escopo da aplicação

O aplicativo será mostrará uma lista de notícias usando a [`NewsApi`](https://newsapi.org/), possibilitando filtragem por data, nome, origem e categoria; ordenação e acessar a notícia no local original e também salvar, na memória do dispositivo, notícias.
